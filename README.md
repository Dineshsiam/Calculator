# Simple Calculator

A basic calculator web application built using HTML, CSS, and JavaScript. It performs simple arithmetic operations with an interactive user interface.


## Features

* Addition, subtraction, multiplication, and division
* Number input buttons
* Clear (AC) functionality
* Delete last input
* Instant calculation
* Simple and responsive UI


## Technologies Used

* HTML – Structure
* CSS – Styling
* JavaScript – Functionality


## Project Structure

```
Simple-Calculator
│── index.html          # UI structure
│── stylesheet.css      # Styling
│── functionality.js    # Logic
```

## How to Run

1. Download or clone the repository
2. Open index.html in your browser
